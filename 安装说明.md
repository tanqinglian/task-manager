# Task Manager 安装说明

## 下载和安装

### 方式 1：图形界面安装（推荐）

1. 下载 `task-manager_0.1.0_aarch64.dmg` 文件
2. 双击打开 DMG 文件
3. 将 `task-manager.app` 拖到 `Applications` 文件夹
4. **首次打开时**：
   - 右键点击应用，选择"打开"
   - 或者按住 Control 键点击应用，选择"打开"
   - 在弹出的安全警告中点击"打开"

### 方式 2：命令行安装

如果你熟悉终端，可以使用以下命令：

```bash
# 1. 挂载 DMG
hdiutil attach task-manager_0.1.0_aarch64.dmg

# 2. 复制应用到 Applications
cp -R /Volumes/task-manager/task-manager.app /Applications/

# 3. 卸载 DMG
hdiutil detach /Volumes/task-manager

# 4. 移除隔离属性（重要！）
xattr -cr /Applications/task-manager.app

# 5. 重新签名（可选）
codesign --force --deep --sign - /Applications/task-manager.app 2>/dev/null
```

## 常见问题

### Q: 为什么应用打不开，显示"已损坏"？

**A:** 这是 macOS 的安全机制。请尝试以下方法：

1. **方法一（推荐）**：右键点击应用 → 选择"打开" → 在弹出的对话框中点击"打开"

2. **方法二**：在终端运行以下命令
   ```bash
   xattr -cr /Applications/task-manager.app
   ```

3. **方法三**：临时关闭 Gatekeeper（不推荐）
   ```bash
   sudo spctl --master-disable
   # 打开应用后，记得重新开启
   sudo spctl --master-enable
   ```

### Q: 可以双击直接打开吗？

**A:** 首次打开需要右键点击"打开"，之后就可以正常双击打开了。

### Q: 这个应用安全吗？

**A:** 是的，这是一个自签名的应用。macOS 会阻止未知的开发者应用以确保安全。通过上述方法打开后，系统会记住你的选择。

## 系统要求

- macOS 10.13 或更高版本
- Apple Silicon (M1/M2/M3) 或 Intel Mac
